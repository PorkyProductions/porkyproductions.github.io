Here at PorkyProductions we are working on a variety of projects. We innovate new technologies everyday, and create new ways to interact with the world around us.

Some of the projects we are working on include:


(de)Motivator

Our flagship product. (de)Motivator is a simple web application that generates a random insult for the user. It is written in TypeScript and HTML + CSS. It is currently being developed by PorkyProductions.
(de)Motivator




PorkyOS

PorkyOS is a new operating system for x86_64 computers. It is currently being developed by PorkyProductions. It is a fully featured operating system, with a fully featured GUI. It is written in C and forked from Linux, so there is no compatibility issues.
PorkyOS




PorkyNet

PorkyNet is a new web browser that will replace what you thought was a web browser. PorkyNet redifines the very definition of a browser, with blazing fast speeds and the features you love.
