Technlogical Developments


Here at PorkyProductions we are working on a variety of projects. We innovate new technologies everyday, and create new ways to interact with the world around us.

We are currently working on a variety of projects, and we are always looking for new ideas to work on.

Some of the projects we are working on include:

PorkyOS

PorkyOS is a new operating system for x86_64 computers. It is currently being developed by PorkyProductions. It is a fully featured operating system, with a fully featured GUI.
It is written in C and forked from Linux, so there should be no compatibility issues.


(de)Motivator

(de)Motivator is a simple web application that generates a random insult for the user. It is written in TypeScript and HTML + CSS.
It is currently being developed by PorkyProductions.


PorkyNet

PorkyNet is a modern web browser that is designed to be a modern replacement for the old web browser. It is written in C++ and uses the Qt framework. It is forked from Chromium.
It is currently being developed by PorkyProductions.
